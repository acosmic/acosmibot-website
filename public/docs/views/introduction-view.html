Introduction

Welcome to the Acosmibot Documentation

These modules walk you through setting up and using Acosmibot features to improve and customize your Discord server. Acosmibot is a feature-rich Discord bot covering server engagement, economy, moderation, streaming alerts, games, and AI.

Key Features

Acosmibot includes the following major features and commands:

Leveling System

Reward server activity with XP and automatic role progression. Members earn experience points by sending messages and level up over time. Supports customizable XP formulas (linear, quadratic, exponential), role rewards at milestones, level-up announcements, daily leaderboards, and XP multipliers. Commands: /rank, /leaderboard, /stats.

Economy and Banking

A virtual currency system using credits. Members earn credits through gambling games, lotteries, and credit transfers. The global bank lets members deposit and protect their credits across all servers. Transaction history and leaderboards track wealth over time. Commands: /balance, /give, /deposit, /withdraw, /bank, /bank-history, /guildbank, /leaderboard.

Social Alerts — Twitch, YouTube, Kick

Real-time streaming notifications when your favorite streamers go live. Supports Twitch (EventSub webhooks), YouTube (PubSubHubbub), and Kick. Each platform supports customizable announcement messages, role mentions (@everyone, @here, or specific roles), and per-streamer configuration.

Games and Gambling

A suite of credit-based gambling games integrated with the economy system: Slots (3-reel machine with customizable symbols and payouts), Lottery (server-wide jackpot drawings), Blackjack (classic card game vs the dealer), Coinflip (50/50 heads or tails), Deathroll (high-stakes dice duels between members). Commands: /slots, /blackjack, /coinflip, /deathroll, /rockpaperscissors, /8ball. Server-wide lottery events run via /admin-start-lotto.

Moderation Logs

Automatic logging of key server events to your designated mod channels. Captures member joins and leaves (distinguishing kicks and bans from voluntary departures), deleted messages with original content, and edited messages showing before and after. Each event type can be routed to a separate channel and toggled independently. Configured through the server dashboard — no slash commands required.

AI Integration

Connects Acosmibot to an AI model so members can have natural conversations by mentioning the bot. Maintains context for the last 15 messages per channel, supports a custom system prompt (configured in the dashboard) to define the AI's personality, configurable daily usage limits, and channel restrictions. Premium servers also get /generate-image (DALL-E 3) and /analyze-image (GPT-4 Vision). Admin commands: /ai-enable, /ai-disable, /ai-status, /ai-clear.

Utilities

Additional tools to enhance your server: Reaction Roles (assign roles via emoji reactions), Custom Commands (create your own slash commands with custom responses), Better Embeds (rich embedded message builder), Reminders (scheduled personal or server reminders).

Chaos Features

Advanced and experimental features: Cross-Server Portals (link channels across different Discord servers), Polymorph (temporarily change member nicknames and avatars), Jail System (restrict members to a designated channel as punishment).

Development & Support

Acosmibot is developed and maintained by a single developer. While we do our best to thoroughly test features, there may still be bugs or edge cases we haven't caught yet. If you run into any issues, we appreciate your patience and feedback. We're committed to actively fixing problems, improving performance, and adding new features over time to make Acosmibot better for everyone.
