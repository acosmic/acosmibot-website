Introduction

Welcome to the Acosmibot Documentation

These modules walk you through setting up and using Acosmibot features to improve and customize your Discord server. Acosmibot is a feature-rich Discord bot covering server engagement, economy, moderation, streaming alerts, games, and AI.

Key Features

Acosmibot includes the following major features and commands:

Leveling System

Reward server activity with XP and automatic role progression. Members earn experience points by sending messages and level up over time. Supports customizable XP formulas (linear, quadratic, exponential), role rewards at milestones, level-up announcements, daily leaderboards, streak tracking, and XP multipliers. Commands: /rank, /leaderboard, /setlevel, /addxp.

Economy and Banking

A virtual currency system using credits. Members earn credits through daily rewards, gambling games, and transfers. The global bank lets members deposit and protect their credits. Includes streak bonuses for consecutive daily claims, transaction history, and richest-member leaderboards. Commands: /balance, /daily, /transfer, /deposit, /withdraw, /richest.

Social Alerts — Twitch, YouTube, Kick

Real-time streaming notifications when your favorite streamers go live. Supports Twitch (EventSub webhooks), YouTube (PubSubHubbub), and Kick. Each platform supports customizable announcement messages, role mentions (@everyone, @here, or specific roles), and per-streamer configuration.

Games and Gambling

A suite of credit-based gambling games integrated with the economy system: Slots (3-reel machine with customizable symbols and payouts), Lottery (server-wide jackpot drawings), Blackjack (classic card game vs the dealer), Coinflip (50/50 heads or tails), Deathroll (high-stakes dice duels between members). Commands: /slots, /lottery, /blackjack, /coinflip, /deathroll.

Moderation Tools

Keep your server safe with a full moderation suite including warnings, mutes, kicks, bans, and a jail system. Supports auto-moderation rules, configurable punishment escalation, mod-log channels, and detailed audit trails. Commands: /warn, /mute, /kick, /ban, /jail.

AI Integration

Connects Acosmibot to an AI model so members can have natural conversations by mentioning the bot. Maintains context for the last 15 messages per channel, supports a custom system prompt to define the AI's personality, configurable daily usage limits, channel restrictions, and swappable AI models.

Utilities

Additional tools to enhance your server: Reaction Roles (assign roles via emoji reactions), Custom Commands (create your own slash commands with custom responses), Better Embeds (rich embedded message builder), Reminders (scheduled personal or server reminders).

Chaos Features

Advanced and experimental features: Cross-Server Portals (link channels across different Discord servers), Polymorph (temporarily change member nicknames and avatars), Jail System (restrict members to a designated channel as punishment).

Development & Support

Acosmibot is developed and maintained by a single developer. While we do our best to thoroughly test features, there may still be bugs or edge cases we haven't caught yet. If you run into any issues, we appreciate your patience and feedback. We're committed to actively fixing problems, improving performance, and adding new features over time to make Acosmibot better for everyone.
