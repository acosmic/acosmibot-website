Polymorph

Spend credits to temporarily change another member's display name in your server.

Overview

Polymorph lets members spend credits to rename another member's server nickname for a configured amount of time. Server admins can enable or disable the feature, set the credit cost, choose how long the nickname lasts, and decide whether members type the nickname or let AI generate one from recent channel context.

Key Details

  • Disabled by default until enabled in the server dashboard
  • Default cost is 10,000 credits, configurable per server
  • Default duration is 60 minutes, configurable per server
  • Admins can choose typed nicknames or AI-generated nicknames
  • 10% of the cost is contributed to the guild vault automatically
  • Cannot be used on the server owner
  • Nicknames follow Discord's 32-character limit
  • The bot must have permission to manage nicknames for the target

Commands

/polymorph <user> [new nickname]
Change the target member's server display name. Provide a nickname to choose it yourself. If admins allow AI random names, you can leave the nickname blank and the bot generates one from recent channel conversation history. The nickname is restored after the configured duration.

Examples:

/polymorph @Clint "Lord of the Pings" /polymorph @Clint

Guild Vault Contribution

Every successful polymorph automatically sends 10% of the configured credit cost into your server's guild vault.

Restrictions & Requirements

When Polymorph Won't Work

  • Server Owner: The server owner cannot be polymorphed by anyone
  • Disabled Feature: Polymorph must be enabled in the server dashboard before members can use it
  • Bot Permissions: Acosmibot must have the "Manage Nicknames" permission and must be ranked above the target member in the server role hierarchy
  • Character Limit: The new nickname must be 32 characters or fewer — Discord's hard limit
  • Insufficient Credits: You must have enough credits in your wallet to cover the full cost before the command goes through

Role Hierarchy Note

Discord enforces a strict role hierarchy for nickname management. Acosmibot can only change the nickname of members whose highest role is ranked below the bot's own role. If you're trying to polymorph a server admin or someone with a high role, make sure Acosmibot's role is positioned above theirs in Server Settings → Roles.

How It Plays Out

1

Choose your target

Run /polymorph @member "nickname" to choose a name, or /polymorph @member to use AI random names when admins have enabled them.

2

Credits are deducted

The configured credit cost is taken from your wallet. 10% flows automatically into the guild vault.

3

Nickname changes instantly

The target member's display name updates immediately across the entire server. The bot posts a confirmation so everyone knows what happened and who's responsible.

4

Nickname restores later

When the duration expires, Acosmibot restores the previous nickname if it has not already been changed by someone else.

Frequently Asked Questions

Q: Can the target member change their nickname back?

A: Yes — unless the server has restricted members from changing their own nicknames. A member can always go to Server Settings and update their own display name after being polymorphed.

Q: Can an admin reverse a polymorph?

A: Admins can change any member's nickname at any time through standard Discord server management tools, regardless of how the name was set.

Q: What happens if someone changes the nickname before the timer ends?

A: Acosmibot skips the automatic restore so it does not overwrite a later admin or member edit.

Q: How does AI choose a nickname?

A: When AI random names are enabled and the command nickname is blank, Polymorph sends recent messages from the command channel to the configured Polymorph model and asks for one nickname that fits Discord's length limit.

Q: Why can't I polymorph the server owner?

A: Discord does not allow any bot or user — regardless of permissions — to change the server owner's nickname. This is a Discord platform restriction.

Q: What counts toward the 32-character limit?

A: All characters in the nickname string, including spaces, punctuation, and emoji (which can count as multiple characters). The bot will reject nicknames that exceed the limit before spending your credits.

Q: Does the guild vault have a cap?

A: The guild vault accumulates contributions from polymorph and other sources. Check the economy section of your server dashboard to see the current vault balance and how it's used.
