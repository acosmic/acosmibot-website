Economy & Banking System

A fully-featured currency system with credits, daily rewards, transfers, and a global bank for your community.

Overview

Acosmibot's economy system provides a virtual currency called "credits" that members can earn, spend, and transfer. The system integrates with games, rewards, and other features to create an engaging server economy.

Key Features

  • Credit wallet per server — earn by playing games and receiving transfers
  • Global bank for safe cross-server storage with deposit and withdrawal history
  • Give credits to other members with /give
  • Server guild bank vault viewable with /guildbank
  • Integration with all gambling games (slots, blackjack, coinflip, deathroll)
  • Leaderboards for Currency, XP, and Level via /leaderboard

How It Works

💰 Credits

Credits are the virtual currency in Acosmibot. Members start with 0 credits and can earn them through:

  • Gambling Games: Win credits by playing slots, blackjack, coinflip, deathroll, etc.
  • Transfers: Receive credits from other members via /give
  • Lottery Winnings: Win credits from server-run lottery events

🏦 Global Bank

The bank allows members to store credits safely. Banked credits cannot be lost in gambling games until withdrawn. This encourages members to save their earnings.

  • Deposits: Store credits in the bank
  • Withdrawals: Take credits out for spending
  • No Interest: Bank balance doesn't grow automatically
  • Safe Storage: Banked credits are protected

Commands

/balance [member]
Check your current credit balance and bank balance. Optionally view another member's balance.

Examples:

/balance /balance @username
/give <member> <amount>
Send credits from your wallet to another member. The amount is deducted from your server balance.

Example:

/give @username 500
/deposit <amount>
Deposit credits from your server wallet into the global bank for safe keeping. Use all to deposit everything. A small fee may apply depending on server settings.

Examples:

/deposit 1000 /deposit all
/withdraw <amount>
Withdraw credits from the global bank back to your server wallet. Use all to withdraw everything.

Examples:

/withdraw 500 /withdraw all
/bank
View your global bank balance, total deposited, total withdrawn, and account statistics.

Example:

/bank
/bank-history
View your recent bank transaction history, including deposits, withdrawals, fees, and dates.

Example:

/bank-history
/guildbank
Check the current credit balance of this server's shared guild vault. The guild vault funds server-wide events like the lottery.

Example:

/guildbank
/leaderboard <stat>
View the top 10 members for a chosen stat. Available stats: Currency, Exp, Level, Global Currency, Global Exp, Global Level.

Examples:

/leaderboard Currency /leaderboard Global Currency

Bank Transaction History

All bank transactions are tracked and viewable with /bank-history, including:

  • Deposits and the fee charged (if any)
  • Withdrawals and timestamps
  • Running balance after each transaction

📊 Viewing Your History

Use /bank-history in Discord to see your recent transactions. Your user dashboard at /dashboard also shows balance history over time.

Best Practices

💡 Tips for Server Owners

  • Run lotteries via /admin-start-lotto to recirculate guild vault credits back into the economy
  • Monitor the /leaderboard Currency to keep an eye on balance distribution
  • Set gambling limits in the dashboard to prevent runaway losses
  • Encourage banking — banked credits can't be lost in gambling games
  • Use giveaways to reward active members with credits

⚠️ Economy Management

  • Don't over-inflate the economy with excessive admin grants
  • Balance game payouts to maintain credit value
  • Consider periodic "wealth taxes" or credit sinks for balance
  • Monitor for exploitation or abuse of transfer/gambling systems

Troubleshooting

Q: Can I transfer credits between servers?

A: Server wallet credits are server-specific. However, the global bank is shared across all servers — you can deposit in one server and withdraw in another.

Q: What happens if I go into negative credits?

A: You cannot go negative. Gambling games and /give will block the action if you don't have enough credits.

Q: My deposit failed

A: Make sure the bank system is enabled on this server (check with an admin). There may also be a minimum deposit amount configured. The error message will tell you the exact requirement.

Q: Where does the guild vault come from?

A: The guild vault accumulates credits from server economy activity. Admins can run lotteries that pay out from the vault to reward community members.
