Command List

A complete reference of every Acosmibot command, organized by category. Use /commands in Discord for the interactive version.

In-Discord Help Menu

The fastest way to explore commands is directly in Discord. Run /commands and the bot will open an interactive paginated menu organized across four categories. Navigation buttons let you flip between pages, and the menu stays active for three minutes.

/commands
Open the interactive command reference menu. Displays all available commands organized across four pages with navigation buttons. The menu stays active for 3 minutes before timing out.

Example:

/commands

Page 1 — Games & Gambling

All credit-based games and entertainment commands.

Permission Levels

Who Can Use What

  • Everyone — Games, economy, bank, reminders, rank, stats, utility (ping, avatar, apod, polymorph)
  • Administrator — AI controls (enable, disable, status, clear), lottery admin, server dashboard settings
  • Premium servers — Image generation and analysis, extended custom commands limit
  • Server Owner — Full access to all commands and dashboard settings

AI — Mention-Based Interaction

The AI feature does not use a slash command. Members interact with the bot by mentioning it: @Acosmibot what is the leveling system?. The bot must be enabled by an admin first using /ai-enable.

Frequently Asked Questions

Q: A command isn't showing up for me — why?

A: Some commands only appear if the feature is enabled in your server. If a command is missing entirely, check that Acosmibot has been authorized with the correct scopes and that the feature is toggled on in the /server dashboard.

Q: How long does the /commands help menu stay open?

A: The interactive menu stays active for 3 minutes. After that the navigation buttons stop working, but you can run /commands again at any time to get a fresh one.

Q: Can I restrict certain commands to specific channels?

A: Yes. Discord's built-in slash command permissions let server admins restrict any command to specific channels or roles — no bot configuration needed. Go to Server Settings → Integrations → Acosmibot to manage this.

Q: Are new commands added over time?

A: Yes. Acosmibot is actively developed and new commands are added regularly. Keep an eye on the changelog or join the support server for announcements.
