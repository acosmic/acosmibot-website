Kick Integration

Get real-time notifications when streamers go live on Kick with webhook-based alerts.

Overview

Similar to Twitch integration, Kick alerts notify your community when streamers go live on Kick.com with customizable messages and role mentions.

Configuration

Set up through /server dashboard → Kick section:

  • Enable Kick integration
  • Set announcement channel
  • Add Kick streamer usernames
  • Configure role mentions per streamer
  • Customize announcement messages

Message Placeholders

{streamer} - Streamer name {username} - Kick username {title} - Stream title {category} - Stream category {url} - Kick channel URL
