Quick Start Guide

Get Acosmibot up and running in your server in just a few minutes with this step-by-step guide.

Step 1: Invite Acosmibot

Click the button below to invite Acosmibot to your Discord server. You'll need the "Manage Server" permission.

Invite Acosmibot to Your Server

Required Permissions

For full functionality, grant these permissions:

  • Read Messages/View Channels - See channels and messages
  • Send Messages - Post announcements and responses
  • Manage Roles - Assign leveling role rewards
  • Embed Links - Send rich embeds
  • Attach Files - Share images and files
  • Mention @everyone, @here, All Roles - For stream alerts
  • Use Slash Commands - Enable command usage

Step 4: Use Commands

Acosmibot uses Discord's slash commands. Type / in any channel to see available commands.

Essential Commands to Try

/help - Show all available commands /rank - Check your level and XP /balance - Check your credits /daily - Claim daily credit reward /leaderboard - View top members

Command Examples

  • /rank - View your current rank and level
  • /balance @user - Check another member's balance
  • /slots 100 - Bet 100 credits on slots
  • /transfer @friend 500 - Send 500 credits to a friend

Step 5: Customize Settings

Recommended First Steps

  1. Set up leveling: Configure XP formula and add role rewards at levels 5, 10, 20, 50
  2. Enable moderation: Create a #mod-logs channel and enable event tracking
  3. Add Twitch alerts: Create a #live-alerts channel and add your favorite streamers
  4. Test commands: Use /daily, /rank, and /slots to test the economy
  5. Create custom commands: Add fun server-specific commands

Common First-Time Setup Questions

Q: Do I need to enable everything?

A: No! Only enable features you want to use. Start with 1-2 features and expand as you get comfortable.

Q: Will enabling leveling retroactively give XP?

A: No, members only start earning XP after the leveling system is enabled. Past messages don't count.

Q: Can members see their stats without using commands?

A: Yes! Members can visit /dashboard to view their profile, stats, and transaction history.

Q: How do I give starting credits to members?

A: Use /givecredits @member <amount> as an admin, or encourage members to use /daily to build up credits.

Q: What if I make a mistake in configuration?

A: All settings can be changed at any time in the dashboard. Changes take effect immediately.
