Quick Start Guide

#1 Inviting Acosmibot

Click the button below to invite Acosmibot to your Discord server. You'll need the "Manage Server" permission to complete the setup.

#2 Granting Permissions

To ensure Acosmibot works as intended, please grant the following permissions:

Core Messaging

  • View Channels — Access channels and read messages
  • Send Messages — Respond to users and post announcements

Roles & Rewards

  • Manage Roles — Assign role rewards for leveling and progression

Content & Formatting

  • Embed Links — Send formatted messages
  • Attach Files — Share images and other files

Notifications

  • Mention @everyone, @here, and All Roles — Send stream and server-wide alerts

#3 Enable Features

When you're in the server dashboard, enable the features you want to use.

Common First-Time Setup Questions

Q: Do I need to turn on the bot every feature?

A: No, you only need to turn on the features you want to use. We suggest starting with one or two and adding more as you get used to the bot.

Q: Why aren't level-up announcements appearing?

A: Check that an announcement channel is selected and that the bot has Send Messages and Embed Links permissions in that channel.

Q: Roles aren't being assigned automatically?

A: Ensure Acosmibot's role is placed higher than the roles it needs to assign. You can adjust this in Server Settings → Roles.

Q: How do I reset levels?

A: For individual members, use /resetxp @user. For bulk or full server resets, please contact support.
