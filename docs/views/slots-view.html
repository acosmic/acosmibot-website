Slots Game

Try your luck at the slot machine! Bet credits and match symbols for big payouts.

Overview

The slots game is a classic slot machine where you bet credits and spin three reels. Match symbols across the reels to win multipliers on your bet.

Key Features

  • 3-reel slot machine with customizable symbols
  • Multiple winning combinations
  • Configurable payout multipliers
  • Visual slot machine display with emojis
  • Integrated with economy system

How to Play

1

Place Your Bet

Use the command /slots <amount> to bet credits.

/slots 100
2

Spin the Reels

The bot spins three reels with random symbols from the configured symbol pool.

3

Check for Wins

If symbols match, you win! Payouts are based on which symbols matched:

  • 3 of a kind: Highest payout (e.g., 10x bet)
  • 2 of a kind: Medium payout (e.g., 2x bet)
  • No match: Lose your bet

Commands

/slots <amount>
Bet the specified amount of credits and spin the slot machine.

Examples:

/slots 50 /slots 1000

Configuration

Server admins can customize the slots game through the /server dashboard:

Setting Description
Enabled Enable/disable the slots game
Symbol Pool Customize the emojis used on the reels (default: 12 symbols)
Payout Rates Adjust multipliers for different winning combinations
