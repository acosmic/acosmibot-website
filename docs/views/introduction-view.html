Getting Started

1

Invite Acosmibot to Your Server

Add Acosmibot to your Discord server using our invite link. Make sure to grant the necessary permissions for full functionality.

Invite Acosmibot
2

Access the Dashboard

Log in with Discord and navigate to your server's configuration dashboard to customize features.

Go to Dashboard
3

Enable Features

Enable and configure the systems you want to use. Each feature can be customized independently via the /server dashboard.

4

Start Using Commands

Acosmibot uses Discord slash commands. Type / in your server to see all available commands.

/help
